# Rapid Relief Blood Platform

Rapid Relief Blood Platform is a web-based system that helps connect blood donors with patients quickly during emergencies.

## Features
- Donor registration
- Blood request system
- Email notifications
- Emergency blood search
- User-friendly interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- MySQL

## Purpose
This platform helps reduce the time required to find blood donors during emergencies.
